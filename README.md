# AI人狼システム

「AIは人狼を遊べるのか？」の探求的検証プロジェクト

## 概要

人狼経験者も満足する高度な戦略性を持つAI人狼システムの研究・開発

### プロジェクトの目的

- 役割別エージェントによる人狼ゲームシミュレーション
- 高度な業務シミュレーションへの応用
- AIエージェント間の協調・対立の研究

## 特徴

- **票数ゲームとしての本質理解**: 論理的必然性による議論支配
- **先手必勝の戦略思考**: 個人信用に依存しない戦術
- **役割別エージェント**: 各プレイヤーが独立したエージェントとして動作
- **拡張性**: 人狼以外の協調ゲームへの応用可能
