# AI人狼システム：導入・運用ガイド

**バージョン**: v1.0  
**最終更新**: 2026-01-23

## 📋 【概要】
本システムは、AIが人狼ゲームを高いレベルでプレイできるようにするための包括的なプロンプトシステムです。人狼ゲーム経験者にも満足いただける戦略的思考力を持つAIプレイヤーの実現を目指しています。
