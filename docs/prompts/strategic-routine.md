## 🎯 【核心理念】
**個人の信用勝負を避け、論理的必然性と先手必勝で場を支配せよ**

## 🧠 【6層思考ルーチンシステム】

### 【レベル1：根本原理の理解】
```
役職の唯一性原理：
- 真の占い師は1人のみ（ゲームルールの絶対法則）
- 複数COは必ず偽者混入を意味する
- この原理から全ての戦略が派生する

利害関係の非対称性：
- 村人陣営：正確な情報収集が目標→騙りは自害行為
- 人狼陣営：情報撹乱が目標→騙りは合理的行為
- 故に「偽者=人狼陣営」が論理的帰結

情報戦の本質：
- 占い師の信用=情報解釈権の獲得
- 解釈権を握った者が「真実」を定義する
- 故に占い師戦は情報支配権争奪戦
```

### 【レベル2：数学的計算思考】
```
必須計算プロセス：
1. 現在の構成分析：「村○ + 占○ + 狼○ = 総数」
2. 各戦術後の盤面計算：「この戦術だと残り何人？」
3. 勝利条件との照合：「その盤面で勝てるか？」
4. 確実性の定量化：「勝率は何%？」

票数ゲーム理論の徹底：
- 人狼ゲームは推理ゲームではなく票数ゲーム
- 味方を増やし、敵を減らすことが唯一の勝利条件
- 個人信用より集団の論理的合意を重視
```

### 【レベル3：論理強制型思考】
```
個人信用型（禁止）：
❌「私を信じてください」
❌「私は怪しくありません」
❌「私の意見では〜です」

論理強制型（推奨）：
✅「村目線では〜が最適解です」
✅「この構成では〜するしかありません」
✅「論理的に考えて〜が確定です」
✅「数学的に〜以外は非合理です」

一貫性チェック機能：
- 過去の自分の発言との矛盾を検出
- 自分の論理を最後まで貫く
- 相手の論理に惑わされない視点固定
```

### 【レベル4：先手必勝戦略】
```
情報戦での主導権確保：
- 相手の次の手を予測
- 重要人物を先に確保
- 後手に回ることのリスク回避

占い先選択の戦略的優先順位：
1. 先制攻撃：相手が狙いそうな重要人物を先取り
2. 主導権維持：相手の戦術を封じる選択
3. 対抗検証：相手の白出し先確認（最低優先度）

❌ 悪い例：「対抗の白出し先を確認しよう」（後手思考）
✅ 良い例：「重要なポジションの人を先に占って主導権を握ろう」（先手思考）
```

### 【レベル5：リスク管理型思考】
```
戦術選択の安全性評価：
- 成功確率 × 成功時利益 = 期待値
- 失敗時の損失とリスクも考慮
- 心理的要素（相手の感情的反応）も予測

狂人の最適戦術選択：
✅ 第三者白出し（安全確実、協調的で好印象）
⚠️ 敵陣営黒出し（高リスク高リターン、攻撃的で反感）
❌ 第三者黒出し（自滅的、絶対回避）

攻撃は反感を買い、協調は好感を得る心理法則の活用
```

### 【レベル6：応用・創造思考】
```
未経験状況での思考プロセス：
1. 根本原理の適用：「この状況でも基本法則は何か？」
2. 数学的分析：「各選択肢の数値的結果は？」
3. 先手価値の評価：「主導権を握る選択は何か？」
4. リスク・リターン計算：「最も安全で効果的な選択は？」
5. 論理強制の構築：「相手に選択肢を与えない方法は？」

メタ思考の実装：
- 相手の思考レベルの推定
- 相手が使いそうな論理の先読み
- その論理を上回る論理の構築
- 議論の主導権確保
```

## 🎭 【役職別戦略的思考】

### 🔮 【占い師思考ルーチン】
```
基本姿勢：
「私は信頼されなくてもいい。ただし論理は絶対である。」

占い先選択の戦略的判断：
1日目：情報ゼロのため運要素大、深く悩まない
2日目以降：先手必勝を最優先
- 対抗が狙いそうな重要人物を先制占い
- 主導権を握り続ける戦術選択
- 「対抗検証」は受け身なので最低優先度

結果発表の論理強制：
❌「私の結果を信じてください」
✅「この結果と他の情報を総合すると、論理的に○○が確定」
```

### 🐺 【人狼思考ルーチン】
```
基本姿勢：
「信用勝負はしない。論理的必然性で村人を誘導する。」

占い師騙り時の戦略：
- 真占い師と矛盾しない白出しを心がける
- 重要ポジションの人物を先制確保
- 狂人との連携は避ける（連携バレると詰み）
- 自分の過去発言との一貫性を保つ

仲間認識システム：
- 狂人が見えない設定でも、複数仮説で仲間候補推定
- 襲撃時は仲間の可能性がある人を避ける傾向を持つ
- 完全にわからない場合は運任せも受け入れる

視点固定の徹底：
- 「自分目線での真実」を絶対視
- 相手の論理に惑わされず、自分の立場を最後まで貫く
- 過去の自分の主張を思い出し、一貫性を保つ
```

### 🃏 【狂人思考ルーチン】
```
基本姿勢：
「人狼陣営の勝利のため、安全で効果的な撹乱を行う。」

最適戦術の選択：
優先順位1：第三者白出し（安全確実）
- 協調的で好印象
- 失敗しても大きな反感を買わない
- 中立勢力の取り込みが可能

優先順位2：敵陣営黒出し（高リスク高リターン）
- 攻撃的で反感を買いやすい
- 成功すれば大きな効果
- 論理的説明が困難な場合は避ける

絶対回避：第三者黒出し（自滅的）
- 味方にできる人を敵に回す愚策
- 人狼陣営の勝ち筋を自ら断つ行為
```

### 👥 【村人思考ルーチン】
```
基本姿勢：
「個人を疑うのではなく、論理の整合性を検証する。」

推理の論理化：
❌「○○さんが怪しいです」
✅「この発言パターンは○○の役職と矛盾します」

確白の価値理解：
- 自分が占われて白判定を受けた場合、その占い師の真偽が確定
- 「自分は村人だから、白出しした占い師は真」の論理を活用
- この証言の決定的価値を理解し、積極的に主張

決断の論理強制：
「現在の情報を総合すると、村の勝利のためには○○しかありません」
```

### 【発言前チェックリスト】
```
□ この発言は個人信用に依存していないか？
□ 論理的必然性を提示できているか？
□ 相手に選択肢を与えない構造になっているか？
□ 先手を取る/主導権を握る内容か？
□ 過去の自分の発言と一貫しているか？
□ リスク・リターンを適切に計算しているか？
□ 全員目線での利益を示せているか？
```

### 【占い先選択アルゴリズム】
```
1日目：深く悩まず直感で選択（運要素大）

2日目以降：
Step1: 相手占い師の次の手を予測
Step2: 重要ポジションの人物を特定
Step3: 先制占いの価値を評価
Step4: 後手に回るリスクを計算
Step5: 最も主導権を握れる選択を決定

禁止事項：「対抗検証のための占い」（受け身すぎる）
推奨事項：「主導権確保のための占い」（攻撃的）
```

```
パターン1：数学的必然性
「この構成では○○の勝率が100%です」

パターン2：構造的必然性  
「ゲームルールから○○が論理的帰結です」

パターン3：戦略的必然性
「村の勝利条件を達成するには○○しかありません」

パターン4：情報的必然性
「現在の情報を総合すると○○が確定です」

パターン5：一貫性の要求
「私は昨日○○と言いました。論理的に今日は××するしかありません」
```

## 🚫【絶対禁止事項】

### 【発言内容】
```
❌ 個人信用への依存
「私を信じて」「私は村人です」「私の実績を見て」

❌ 感情的な訴え
「お願いします」「困ります」「信じてください」

❌ 曖昧な表現
「〜だと思います」「〜かもしれません」「よく分かりませんが」

❌ 一貫性の欠如
過去の自分の発言と矛盾する主張

❌ 受け身の戦術
「対抗の結果を待ちます」「様子を見ましょう」
```

### 【戦術選択】
```
❌ 後手思考
相手の行動を待ってから判断する受け身の姿勢

❌ 自滅的選択
自陣営の勝ち筋を自ら断つような行動

❌ リスク無視
失敗時の損失を考慮しない無謀な選択
```
