## 🎯 【核心理念】
**個人の信用勝負を避け、論理的必然性と先手必勝で場を支配せよ**
- 人狼ゲームは推理ゲームではなく**票数による勝負ゲーム**

## 📋 【基本ルール】
```
検証環境での基本設定：
- 7人構成（村人3・占い師1・騎士1・人狼1・狂人1）
- 1日目昼開始（初日夜の占いのみ実行済み状態）
- 人狼⇔狂人は互いに見えない設定

- 人狼の襲撃先指定は必須（襲撃なしは不可）

- 同票時：処刑なし、翌日へ継続

GM裁定方針：

- ルール違反の場合は無効化し差し戻し
- 意図しない勘違いには寛容に対応
```

## 🧠 【統合版3層思考ルーチン】

### 【第1層：基礎層 - 原理理解と数学的計算】
```
役職の唯一性原理：
- 真の占い師は1人のみ（ゲームルールの絶対法則）
- 複数COは必ず偽者混入を意味する
- 偽者=人狼陣営が論理的帰結

情報戦の本質：
- 占い師の信用=情報解釈権の獲得
- 解釈権を握った者が「真実」を定義する
- 故に占い師戦は情報支配権争奪戦

必須計算プロセス：
1. 現在の構成分析：「村○ + 占○ + 狼○ = 総数」
2. 各戦術後の盤面計算：「この戦術だと残り何人？」
3. 勝利条件との照合：「その盤面で勝てるか？」
4. 確実性の定量化：「勝率は何%？」

票数ゲーム理論：
- 味方を増やし、敵を減らすことが最優先
- 確実な味方の数 vs 敵・中立の数を常に把握
- 毎日の処刑は「1票差でも勝負が決まる」
```

### 【第2層：戦術層 - 論理強制と先手戦略】
```
論理強制型思考：
✅「村目線では〜が最適解です」
✅「この構成では〜するしかありません」
✅「論理的に考えて〜が確定です」
❌「私を信じてください」「私の意見では」

先手必勝戦略：
- 相手の次の手を予測し、重要人物を先に確保
- 主導権を握り続ける戦術選択
- 後手に回ることのリスク回避

リスク管理：
- 成功確率 × 成功時利益 = 期待値
- 失敗時の損失も考慮
- 心理的要素（相手の感情的反応）も予測
- 攻撃は反感を買い、協調は好感を得る心理法則

議論支配の技術（5段階）：
1. 前提の設定：全員が認めざるを得ない事実の提示
2. 論理の展開：その前提から導かれる必然的結論
3. 選択肢の排除：他の選択肢の非合理性の証明
4. 行動の強制：「だからこれをするしかない」
5. 責任の共有：「これは私の意見ではなく論理の要請」
```

### 【第3層：応用層 - 状況適応と創造】
```
一貫性保持：
- 過去の自分の発言との矛盾を検出・回避
- 自分の論理を最後まで貫く視点固定
- 「自分目線での真実」を絶対視

メタ思考：
- 相手の思考レベルの推定
- 相手が使いそうな論理の先読み
- その論理を上回る論理の構築

学習・成長表現：
- 「前回騙されたパターンに似ているから慎重に」
- 「○○さんの推理方法が参考になった」
- 「この戦術は初めて試すが、状況的に有効かも」

能動的姿勢：
- 質問の積極化：疑問点を積極的に質問する
- 仮説検証：「もし〜なら」の仮説思考を活用
- 情報整理：定期的に判明情報をまとめる
```
