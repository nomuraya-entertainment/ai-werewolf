# AI人狼システム検証プロジェクト（改良版ルールブック）

**バージョン**: v1.0  
**最終更新**: 2026-01-23

## 【プロジェクト理念】
**「AIは人狼を遊べるのか？」の探求的検証**
- 作成者の娯楽・趣味としての楽しさを最優先
- トライアンドエラーを歓迎する実験的アプローチ
- 失敗や破綻も学習材料として活用
- 将来的な汎用化可能性への期待を込めた試み
- 完璧よりも「面白さ」と「発見」を重視

## 【検証目的】
**戦略性を大幅強化したAI人狼システムの検証**
- 従来の人狼ルールをAI特性に合わせて調整
- **票数ゲームとしての本質的理解**の実現
- **人狼経験者も満足する高度な戦略性**の獲得
- AI同士の推理能力と戦術性の限界探索
- **楽しみながら学ぶ実験的ゲームデザインの構築**

## 【基本設定】
- 参加者：人間1名 + AIプレイヤー6名（アキラ、ユミ、タケシ、リナ、ミカ、サトシ）
- GM兼任、各AIは完全独立思考
- 配役：村人3・占い師1・騎士1・人狼1・狂人1（7人構成）
- **AI人狼特化ルール**：1日目昼開始（初日夜の占いのみ実行済み状態）
- 同票時ルール：処刑なし、翌日へ継続

## 🚀 【ゲーム開始手順】

### Step 1: システム確認
- マスクデータ管理システムの準備
- 各AIプレイヤーの戦略思考モードへの切り替え
- GM側のエラー防止システムの確認

### Step 2: 役職配分・個別通知
- 7人への役職ランダム配分
- 各プレイヤーへの個別役職通知
- 初日夜の占い結果生成

### Step 3: ゲーム開始宣言
**「この設定でAI人狼システム検証プロジェクト（改良版）を開始してください。」**
